## 🚀 Features

HalKhata CRM provides a structured system for managing business operations.

### 📇 Contact & Lead Management
* Add, update, and delete customer information  
* Store phone number, address, and interaction history  
* Maintain organized customer records  

---

### 🛒 Order Management System
* Create and manage customer orders  
* Track order status (Pending, Completed, Cancelled)  
* Link orders directly to customers  

---

### 💬 Social-Commerce Support
* Convert conversations into structured orders  
* Maintain message history per customer  
* Simulate unified inbox for business communication  

---

### 📊 Dashboard & Analytics
* Overview of total customers and orders  
* Basic business insights  
* Activity tracking  

---

### 🔐 Authentication & Security
* User login system  
* Role-based access (Admin/User - optional)  
* Secure backend using Spring Boot  

---

## 🏗️ Architecture

This project follows a layered backend architecture using Java Spring Boot:
* **Controller Layer** → Handles API requests  
* **Service Layer** → Business logic  
* **Repository Layer** → Database interaction  
* **Model Layer** → Entity definitions

## 🛠️ Tech Stack

* **Backend:** Java Spring Boot  
* **Database:** PostgreSQL  
* **ORM:** JPA / Hibernate  
* **Build Tool:** Maven
* **Frontend:** Not implemented (API-based backend, HTML/CSS)  
* **Containerization:** Docker
